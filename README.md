# lunz 轮子

> github + npm工程化协作开发栈 脚手架工具

## 快速开始

```
# 利用npx直接使用lunz
npx lunz my-project-name
# 或者全局安装lunz后初始化
npm i -g lunz
lunz my-project-name
```

## 一起加入，构造更完美的最佳实佳！

1. 点击右上角的Fork按钮。
2. 新建一个分支:`git checkout -b my-new-feature`
3. 上报你的更新：`git commit -am 'Add some feature'`
4. 分支上传云端：`git push origin my-new-feature`
5. 提交 `pull request`😘
